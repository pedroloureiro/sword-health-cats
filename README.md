# Sword Health - Cats coding challenge

A coding challenge that consists in creating an application using Kotlin that utilises the Cat API (https://thecatapi.com/)

# secrets.properties
Requires a secrets.properties file on the root of the project to work, with the following keys/values:
```
API_KEY=<insert_api_key>
BASE_URL=https://api.thecatapi.com/
```
